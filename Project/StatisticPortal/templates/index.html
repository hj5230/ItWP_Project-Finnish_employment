StatisticPortal
  • Click "Employment" or "Pensioner" in the navbar to jump to the page showing employment or pensioner api data analysis
  • In Employmnet or Pensioner page:
    • Chart:
      • data in the chart shows employment or pensioner data for designated area, designated gender, designated age
      • drag labels in the header into the chart to change designated gender or age for employment or pensioner data in the chart
      • switch municipality by click a municipality in the map or enter a municipality in the input box and click "Find"
    • Map:
      • mouse over municipality area to view employment or pensioner data, switch the layer to view data for different year
      • the higher the proportion of regional employment or pensioner compare with previous year, the closer the color is to green
      • click municipality area to switch data in the chart
    • Click "Download" to save data in the chart as a PNG file
  • In this project work, Django backend and database are not used
  • Following APIs are used:
    • Frappe-Chart
    • Leaflet
    • Bootstrap
    • html2canvas
